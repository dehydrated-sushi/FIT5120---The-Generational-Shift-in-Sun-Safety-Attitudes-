# Tech Stack

| Layer | Technology | Why |
|---|---|---|
| Frontend | React (Vite) | Dynamic UI, component-based, Recharts integration |
| Backend | Flask (Python) | Team knows Python, lightweight, good for data processing |
| Database | PostgreSQL (Neon.tech) | Industry standard, works with Flask, tutor requirement |
| UV Data | OpenWeatherMap API | Real-time UV index by location |
| Charts | Recharts | React-native, easy to use |
| Animations | Framer Motion | Smooth UI animations |
| Auth | JWT | Industry standard token auth |
| Frontend Host | Vercel | Free, fast, auto-deploys from GitHub |
| Backend Host | Render | Free tier, works with Flask |
| CI/CD | GitHub Actions | Auto deploy on push |
